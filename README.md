# 🧠 Jogo da Memória

Jogo da memória feito com **HTML, CSS e JavaScript puro** (vanilla JS), com efeito de flip 3D nas cartas e sistema de derrota por número de erros.

## 🎮 Como jogar

- Clique em duas cartas para revelá-las.
- Se os emojis forem iguais, o par permanece revelado.
- Se forem diferentes, as cartas voltam a ficar viradas para baixo.
- Você tem no máximo **5 erros**. Ao atingir o limite, o jogo termina e uma mensagem de derrota é exibida.
- Encontre todos os pares antes de errar demais para vencer!

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3 (transformações 3D, gradientes, transições)
- JavaScript (manipulação de DOM, lógica de jogo)

## ▶️ Como rodar o projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/gomes0609/Jogo-da-Memoria.git
   ```
2. Entre na pasta do projeto:
   ```bash
   cd Jogo-da-Memoria
   ```
3. Abra o arquivo `index.html` no navegador (ou use a extensão **Live Server** do VS Code).
